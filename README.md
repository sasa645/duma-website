# DUMA Website

Site da equipe DUMA Community - "Forma, Informa, da Forma"

## 📋 Estrutura

- `index.html` - Página principal com a equipe
- `images/` - Pasta com as fotos dos membros
- `styles/` - (Opcional) Arquivos CSS separados
